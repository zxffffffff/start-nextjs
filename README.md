# start-nextjs
 一个 React + Next 全栈脚手架

# Next.js 介绍
参考：https://nextjs.org/docs

```bash
# 创建项目
npx create-next-app@latest
# 项目说明
app	    应用路由器
pages	页面路由器
public	要提供的静态资产
next.config.js	Next.js 的配置文件
.eslintrc.json	ESLint 的配置文件
next-env.d.ts	Next.js 的 TypeScript 声明文件
tsconfig.json	TypeScript 的配置文件
```
